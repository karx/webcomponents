## <click-clipboard>
Copies to clipboad onclick to the element.

## Use
The string copied to clipboard is calculated as follows.

String to Clipboad = `pre` + `clip` + `innerHtml` + `post`
   
      

Where,

```

<click-clipboard pre=`pre` clip=`clip` post=`post`>
`innerHtml`
</click-clipboard>
```
